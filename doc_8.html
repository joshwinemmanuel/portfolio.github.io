Bag Metadata Mining & Dashboard

This project involves processing over 300,000 sensor bag files to extract and visualize rich spatial-temporal metadata. Each bag file contains embedded GPS and timestamp data, which are mined to build a comprehensive database.

Pipeline Overview

The pipeline automates the extraction and enrichment process using the following steps:

  • Extract latitude, longitude, timestamp, MD5 hash from each bag
  • Calculate time of day (dawn, dusk, day, night) using the astral Python library
  • Use OpenStreetMap API to determine if the location is on a highway, rural area, or urban zone
  • Fetch climate and country details via internal metadata API

Database & Hosting

The extracted and enriched data is stored in a locally hosted database. A custom dashboard interface is developed for monitoring bag processing status and visualizing geographic distributions.
