Motion Prediction Framework (LSTM-based)

A modular framework for LSTM-based motion trajectory prediction. It is designed to work with multiple datasets including Argoverse and other compatible formats. The project is containerized using Docker to ensure reproducibility across environments.

Project Structure


MOTION-PREDICTION-FRAMEWORK/

│   ├── datasets/

│   └── utils/
└── data/

Environment & Execution

The framework is deployed using Docker and includes full support for GPU acceleration: 


cd FMW_Docker
docker build --network host -t motion_pred_lstm .
docker run --gpus all -it --network=host -v <project_dir>:/workspace motion_pred_lstm

Data Support

The framework supports Argoverse as well as other structured datasets. It expects the data to be organized into train/val/test folders for seamless integration during training and evaluation. 


data/
├── argoverse/
│   ├── train/
│   ├── val/
│   └── test/
├── inhouse/
│   ├── train/
│   ├── val/
│   └── test/

Training & Testing

The YAML configuration allows flexibility for hyperparameter tuning and dataset switching. Training and testing are triggered via simple CLI commands: 


python main.py <config.yaml> --gpus 0
python main.py <config.yaml> --gpus 0 --test --load_from <checkpoint.pth>
