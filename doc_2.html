Creating the CAM

took the completed .dxf file to the machine. On the ShopBot's desktop, I opened V Carve Pro and made the necessary changes to the job setup, making sure that use offset was unchecked(Fig.1 in slider). I then imported the .dxf file and pined the tool path side toolbar(Fig.2 in slider). in the toolpath operations, choose the 2D Profile toolpath > Select the cutting parameters > tool accordingly under the 2D Profile toolpath > Configure the Tab and Ramp settings > Now press the calculate button(Fig.3 in slider). A 3D viewport will open, allowing us to visualize the tool path and simulate the CAM(Fig.6 in slider). Click Save in the Toolpath operation to export(Fig.8 in slider). Click the Save Toolpath(s) to File. CAM is created now we can look at how to use the machine control system

Parameters

- cutting depths as 12
- Tool Bit: 6mm Dia
- Step Over: 4.8mm
- Pass Depth: 5mm
- Speed: 1200RPM

Setting Machine and Cutting

cleaned the bed and placed the ply over the bed fixed the ply using screws and seted the orgin using the control software.

By clicking the cut part, you can import the G Code we created earlier. and after importing by keeping default settings click START. Now, turn on the spindle by pressing the start button on the control remote and selecting ok from the pop-up dialogue box on the desktop.
