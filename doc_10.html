Data to ROSBag / Argoverse Format Converter

This tool converts raw automotive `.mf4` log files into formats widely used in autonomous vehicle research: ROSBag and Argoverse. It enables efficient transformation, visualization, and map-based analysis for data engineers and researchers.

Functionality

  • Convert `.mf4` logs to `.bag` for ROS pipelines
  • Extract GPS, object detections, and sensor data into `.csv` chunks (5 sec)
  • Generate Argoverse-formatted data (track ID, timestamp, object type, city)
  • Generate road network maps as `.xml` compatible with Argoverse formatusing coordinates and OSM API
