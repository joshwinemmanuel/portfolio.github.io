Masking and labeling SAR dataset

This project focuses on the segmentation of Synthetic Aperture Radar (SAR) images using neural networks. The dataset of 750+ `.mat` files containing both camera data and image amplitude in a 20log10 scale. The amplitude data is converted into an image and then masked for further processing using neural network-based segmentation models.

Data Labeling and Class RGB Mapping

Manual labels were created using the open-source Computer Vision Annotation Tool (CVAT). The labeled dataset contains various classes such as "Environment", "Street", "Car", and "Pedestrian", which are assigned specific RGB values for segmentation purposes.

Class RGB Colour
Environment 255,102,102
Street 0,0,0
Barrier 221,255,51
Bike 250,250,55
Car 255,0,204
Pole 115,51,128
Vegetation 102,255,102
Pedestrian 255,96,55

Segmentation Output

Neural Network-based segmentation mask

The neural network-based segmentation model was trained on these labeled images to identify and classify the objects in the SAR images. The segmented images can be used for further analysis or for improving autonomous systems that rely on SAR imaging.
